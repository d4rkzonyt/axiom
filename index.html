SANDBOXED
BY DESIGN

üîê
Context Isolation

Renderer processes are fully isolated from the main process. No shared globals, no cross-context contamination.

contextIsolation: true
üö´
No Node in Renderer

Node.js integration is disabled in all renderer processes. Web content cannot access the filesystem or OS APIs.

nodeIntegration: false
üì¶
Sandboxed Renderers

Every BrowserView runs in a sandboxed renderer with Chromium's process isolation and site isolation enabled.

sandbox: true
üõ†Ô∏è
No DevTools in Production

DevTools are disabled in all production builds. Only accessible in development via explicit flag. Ships clean.

devTools: process.env.DEV only
‚ö°
Hardware Acceleration

GPU compositing enabled by default. Chromium's hardware-accelerated rendering pipeline for smooth 60fps+ UI.

--enable-gpu-rasterization
üïê
Background Timer Limiting

Background tabs have their JavaScript timers throttled to 1Hz minimum, reducing CPU wake-ups by up to 90%.
